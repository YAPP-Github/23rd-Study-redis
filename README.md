- **스터디 시간 전까지**!! 공부한 것을 md, pdf등의 형식으로 정리한 뒤, PR을 날려주세요.
  - 브랜치, 커밋 메시지는 {주차}_{이름} 형식을 지켜주세요. (ex week01_yubin)
